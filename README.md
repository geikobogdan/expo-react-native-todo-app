# Welcome to your Expo app 👋

This is an [Expo](https://expo.dev) project created with [`create-gluestack-app`](https://www.npmjs.com/package/create-gluestack).

## Get started

1. Start the app

   ```bash
    npm run start
   ```

2. Start the JSON server

   ```bash
    npm run json-server
   ```

   3. Code Quality

   ```bash
    npm run beautify
   ```
